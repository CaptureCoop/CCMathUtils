# CCUtils
Multi purpose library

## How to use with Gradle:

build.gradle:
```
dependencies {
    implementation 'net.capturecoop.ccutils:ccutils:VERSION'
}
```
settings.gradle:
```
sourceControl {
    gitRepository("https://github.com/CaptureCoop/CCUtils.git") {
        producesModule("net.capturecoop.ccutils:ccutils")
    }
}
```
