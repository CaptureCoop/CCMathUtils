# CCMathUtils
Math Utils library

## How to use with Gradle:

build.gradle:
```
dependencies {
    implementation 'net.capturecoop.ccmathutils:CCMathUtils:VERSION'
}
```
settings.gradle:
```
sourceControl {
    gitRepository("https://github.com/CaptureCoop/CCMathUtils.git") {
        producesModule("net.capturecoop.ccmathutils:CCMathUtils")
    }
}
```
